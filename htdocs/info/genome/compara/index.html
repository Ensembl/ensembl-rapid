Comparative analyses

Homologues

All rapid release species now have homologues with an appropriate set of reference species, allowing us to assign gene symbols where possible. In order to allow us to release this data in a timely manner across hundreds of species, we create one compara database per species, and these are periodically updated as new annotations become available for their references species.
