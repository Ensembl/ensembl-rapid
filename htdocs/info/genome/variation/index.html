An overview of variant annotation

Human Pangenome

  • We display variation data from gnomAD Genomes v3.1.2 and ClinVar 2022-07-09 (lifted over from GRCh38 using crossmap and the chain file in the genome browser. Both variant sets can be downloaded via the FTP in VCFs, with molecular consequences predicted by Ensembl VEP
  • Additionally, we have generated an Ensembl VEP cache containing the remapped Ensembl/GENCODE gene set to facilitate analysing your own variant data with Ensembl VEP. This is also available via the FTP. The gnomAD and ClinVar data described above can be used in VEP analysis using the custom annotations option

Human T2T-CHM13v2.0 (GCA_009914755.4)

  • For the T2T-CHM13v2.0 genome assembly we display the following tracks produced by our colleagues in the Telomere to Telomere (T2T) consortium, with added molecular consequences predicted by Ensembl VEP, in addition to tracks for gnomAD and ClinVar:
    • dbSNP build 155 (lifted over)
    • NHGRI-EBI GWAS catalog (lifted over)
    • 1000 Genomes Phase 3 (recalled on CHM13)
    • Simons Genome Diversity Project (recalled on CHM13)
  • The annotated VCFs are all available from our FTP
